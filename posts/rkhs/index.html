Building up RKHSs from the Riesz Representation Theorem

Motivation

Reproducing Kernel Hilbert Spaces (RKHSs) are a major deal in machine learning, especially in the study of kernel methods and SVMs. RKHSs are a natural extension of Hilbert spaces to function spaces and give us several useful results in statistical learning theory. We'll first prove the Riesz Representation Theorem and then use it to prove the representer theorem for RKHSs, and show that any kernel has an RKHS.

The Riesz Representation Theorem (for Hilbert Spaces)

Statement

Let H\mathcal{H}H be a Hilbert space. Then for every continuous linear functional f∈H∗f \in \mathcal{H}^*f∈H∗ (the dual space of H\mathcal{H}H), there exists a unique element v∈Hv \in \mathcal{H}v∈H such that f(u)=⟨u,v⟩f(u) = \langle u, v \ranglef(u)=⟨u,v⟩ for all u∈Hu \in \mathcal{H}u∈H.

Parsing this statement is a bit intimidating, especially without a bit of background in linear algebra or functional analysis. Essentially, the RRT states that every function fff that takes a vector u∈Hu \in \mathcal{H}u∈H and returns a scalar in a way that is linear and continuous is always expressible as the inner product of that vector with some fixed other vector v∈Hv \in \mathcal{H}v∈H, and this vector vvv is unique and thus defines the linear functional fff.

Reproducing Kernel Hilbert Spaces

Kernels

Before diving into RKHSs, we must first define a kernel. A kernel acts as a similarity measure between data points. More precisely, a kernel is a function K:X×X→RK : X \times X \to \mathbb{R}K:X×X→R (or C\mathbb{C}C) such that, for any pair of inputs x,x′∈Xx, x^\prime \in Xx,x′∈X, the value K(x,x′)K(x, x^\prime)K(x,x′) gives some notion of similarity between xxx and x′x^\primex′.

A kernel must satisfy the following two properties:

  • Symmetry: K(x,x′)=K(x′,x)K(x, x^\prime) = K(x^\prime, x)K(x,x′)=K(x′,x)
  • Positive semi-definiteness: For any finite set of points x1,…,xn∈Xx_1, \dots, x_n \in Xx1​,…,xn​∈X, the kernel matrix Kij:=K(xi,xj) K_{ij} := K(x_i, x_j) Kij​:=K(xi​,xj​) must be symmetric and positive semi-definite. In other words, for any c1,…,cn∈Rc_1, \dots, c_n \in \mathbb{R}c1​,…,cn​∈R, ∑i,j=1ncicjK(xi,xj)≥0. \sum_{i,j=1}^n c_i c_j K(x_i, x_j) \geq 0. i,j=1∑n​ci​cj​K(xi​,xj​)≥0.

Kernels can be thought of as implicitly defining a feature space F\mathcal{F}F and a mapping ϕ:X→F\phi : X \to \mathcal{F}ϕ:X→F s.t. K(x,x′)=⟨ϕ(x),ϕ(x′)⟩F. K(x, x^\prime) = \langle \phi(x), \phi(x^\prime) \rangle_{\mathcal{F}}. K(x,x′)=⟨ϕ(x),ϕ(x′)⟩F​. Kernels are useful since we don't need to know ϕ\phiϕ explicitly (often referred to as the kernel trick). Via kernels, we may compute inner products in high or infinite dimensional feature spaces without ever leaving our original input space.

RKHSs

Now we can define a Reproducing Kernel Hilbert Space. An RKHS is a Hilbert space of functions where evaluation at a point is a continuous linear operation and this evaluation can be "reproduced" via an inner product with a kernel function.

Formally, let XXX be a nonempty set and H\mathcal{H}H be a Hilbert space of functions f:X→Rf: X \to \mathbb{R}f:X→R. Then, HHH is a reproducing kernel hilbert space if there exists a function K:X×X→RK: X \times X \to \mathbb{R}K:X×X→R so that

  • For every fixed x∈Xx \in Xx∈X, K(x,⋅)∈HK(x, \cdot) \in \mathcal{H}K(x,⋅)∈H and
  • ∀f∈H,x∈X,f(x)=⟨f,K(x,⋅)⟩H\forall f \in \mathcal{H}, x \in X, f(x) = \langle f, K(x, \cdot) \rangle_\mathcal{H}∀f∈H,x∈X,f(x)=⟨f,K(x,⋅)⟩H​

The second property (the reproducing property) is the more interesting one that we really care about. It effectively tells us that evaluating a function is the same as taking an inner product between the function and a specific element of the space (K(x,⋅)K(x, \cdot)K(x,⋅)). Equivalently, it says that the kernel functions serve as "representatives" of point evaluation.

This result allows us to apply tools from Hilbert space theory to function evaluation and learning. Since kernels are defined without reference to a specific feature space, we can build function spaces with nice properties (e.g. smoothness, regularity) by choosing an appropriate kernel.

Representer Theorem

Now, we introduce the Representer Theorem, one of the most important results from statistical learning theory. In a nutshell, it explains why kernel methods are tractable, despite involving very high or infinite dimensional function spaces.

Setup

Let's assume we wish to solve a problem of the form min⁡f∈H∑i=1nL(yi,f(xi))+λ∥f∥H2. \min_{f \in \mathcal{H}} \sum_{i=1}^n L(y_i, f(x_i)) + \lambda \left\| f \right\|^2_\mathcal{H}. f∈Hmin​i=1∑n​L(yi​,f(xi​))+λ∥f∥H2​. Here, H\mathcal{H}H is a reproducing kernel Hilbert space with kernel KKK, {(xi,yi)}i=1n\{(x_i, y_i)\}_{i=1}^n{(xi​,yi​)}i=1n​ are our training data, L:R×R→RL: \mathbb{R} \times \mathbb{R} \to \mathbb{R}L:R×R→R is a loss function, and λ>0\lambda > 0λ>0 is a regularization parameter penalizing the complexity of the function fff.

Statement

Let H\mathcal{H}H be a reproducing kernel hilbert space with kernel KKK and suppose the objective depends on fff only via its evaluations at the training points and its RKHS norm. Then, any minimizer f<em>∈Hf^<em> \in \mathcal{H}f<em>∈H can be represented in the form f</em>=∑i=1nαiK(xi,x) f^</em> = \sum_{i=1}^n \alpha_i K(x_i, x) f</em>=i=1∑n​αi​K(xi​,x) for coefficients α1,…,αn∈R\alpha_1, \dots, \alpha_n \in \mathbb{R}α1​,…,αn​∈R. Thus, instead of searching over all function space, we need only search over the nnn coefficients α1,…,αn\alpha_1, \dots, \alpha_nα1​,…,αn​, and we have reduced an infinite dimensional problem into a finite dimensional one.

Moore-Aronszajn Theorem

So far, we've shown that RKHSs have several nice properties. Now, we show that they are guaranteed to exist for a kernel.

Statement

Let K:X×X→RK: X \times X \to \mathbb{R}K:X×X→R be an SPSD (symmetric positive semi-definite) function. Then, there exists a unique RKHS HK\mathcal{H}_KHK​ of functions f:X→Rf: X \to \mathbb{R}f:X→R such that

  • For every fixed x∈Xx \in Xx∈X, K(x,⋅)∈HKK(x, \cdot) \in \mathcal{H}_KK(x,⋅)∈HK​ and
  • ∀f∈HK,x∈X,f(x)=⟨f,K(x,⋅)⟩HK\forall f \in \mathcal{H}_K, x \in X, f(x) = \langle f, K(x, \cdot) \rangle_{\mathcal{H}_K}∀f∈HK​,x∈X,f(x)=⟨f,K(x,⋅)⟩HK​​

In other words, we don't need to start with a function space and find a kernel the kernel itself builds its own Reproducing Kernel Hilbert Space.

Construction

The RKHS HK\mathcal{H}_KHK​ can be explicitly constructed from the kernel KKK as follows:
